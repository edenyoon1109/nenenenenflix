# 🎬 넷플릭스 콘텐츠 장르 분석 (Streamlit)

이 프로젝트는 넷플릭스 공개 데이터를 바탕으로 콘텐츠 장르, 국가, 제작 연도 트렌드를 시각화하는 Streamlit 웹앱입니다.

## 📊 사용 데이터
- 출처: [Kaggle Netflix Dataset](https://www.kaggle.com/datasets/shivamb/netflix-shows)
- 파일: `netflix_titles.csv`

## 🛠️ 사용 도구
- Python, Pandas
- Plotly, Streamlit
